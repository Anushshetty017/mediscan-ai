#  MediScan AI

AI-powered web app for medical image classification.

##  Features

* Brain Tumor Detection
* Skin Cancer Classification
* Fast predictions using EfficientNet
* Simple and clean UI

## Tech Stack

* Python (Flask)
* PyTorch
* HTML, CSS

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

##  Note

This project is for educational purposes only and not for medical diagnosis.
